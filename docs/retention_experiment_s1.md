## 3) 실험 목록

### EXP-01 온보딩 레벨 테스트 간소화
- 가설:
  - 온보딩 질문 수를 줄이면 초기 이탈이 감소하고 첫 학습 진입률이 상승한다.
- 실험 설계:
  - A안(기존): 레벨 질문 8문항
  - B안(실험): 레벨 질문 3문항 + 즉시 학습 진입
- 주요 지표:
  - 온보딩 완료율
  - 첫 학습 시작률(`lesson_started`)
  - 첫 학습 완료율
- 실패 기준:
  - B안의 첫 학습 완료율이 A안 대비 5%p 이상 하락 시 실패

### EXP-02 Streak 리워드 노출 시점 최적화
- 가설:
  - Streak 리워드를 세션 시작 전에 보여주면 재방문 동기가 높아진다.
- 실험 설계:
  - A안: 세션 완료 후에만 streak 노출
  - B안: 홈 진입 시 오늘 streak 목표 먼저 노출
- 주요 지표:
  - D1 Retention
  - D7 Retention
  - 주간 세션 수
- 실패 기준:
  - B안의 D7 Retention이 A안 대비 개선 없고, 세션 수가 감소하면 실패

### EXP-03 복습 푸시 타이밍 최적화
- 가설:
  - SRS 만료 직후(30분 이내) 푸시가 고정 시간 푸시보다 복습 완료율을 높인다.
- 실험 설계:
  - A안: 매일 20:00 고정 푸시
  - B안: 사용자별 `next_due_at` 기준 동적 푸시
- 주요 지표:
  - 푸시 오픈율
  - 복습 완료율(`srs_review_done`/푸시 수신자)
  - D7 Retention
- 실패 기준:
  - B안 오픈율이 A안 대비 개선 없고, 복습 완료율이 3%p 이상 하락 시 실패

### EXP-04 쉐도잉 재시도 유도 카피 A/B
- 가설:
  - 피드백 카피를 행동형으로 바꾸면 쉐도잉 재시도율이 상승한다.
- 실험 설계:
  - A안: 중립 카피
  - B안: 행동 유도 카피(예: "호흡을 짧게 끊어 다시 해보세요")
- 주요 지표:
  - 쉐도잉 재시도율
  - 세션 내 쉐도잉 횟수
- 실패 기준:
  - B안 재시도율 증가가 없고 세션 이탈률이 상승하면 실패
